# AI-Driven Natural Disaster Early Warning Platform

##  Project Overview
The **AI-Driven Natural Disaster Early Warning Platform** is a web-based application that provides **real-time weather insights and early warning signals** for natural disasters such as floods, cyclones, heatwaves, and extreme weather conditions.

The platform integrates **public weather APIs** and presents data through a **simple, responsive, and user-friendly interface**, helping users take timely preventive action.

---

## Key Features
-  Real-time weather updates by city search  
-  Working weather search engine  
-  Early warning indicators based on weather data  
-  Responsive and clean UI  
-  AI-ready architecture for future predictions  
-  API-driven data handling (JSON)
